# First Year Statistics Project Analysis 
This project involved analysis of the annual UK Labour Force Survey. The null hypothesis was that there was no association between the net weekly pay (NETWK) and life satisfaction (SATIS) variable. 
We utilised both an ANOVA and contingency table test for the initial test, than began to perform the same tests but now conditioning on additional variables such as age and gender.
The main weakness of the analysis is that the NETWK variable is capped at £788/wk, and this skews the distribution of NETWK to the right. Despite this limitation, we saw the sort of results that we expected.
